# 都市传说悬疑惊悚小说

## 主题
在城市的阴影中，有些传说不仅仅是故事...

## 视角
第一人称"我"

## 风格
悬疑、惊悚、心理恐怖
都市传说背景
开放式结局

## 章节（10章）
1. 听说：主角听到一个都市传说
2. 不信：主角认为只是故事
3. 初现：开始发生诡异事件
4. 调查：主角试图找出真相
5. 深入：发现传说的根源
6. 逼近：恐怖越来越近
7. 真相：可怕的真相浮出水面
8. 绝望：主角陷入绝境
9. 抉择：必须做出选择
10. ？？？：开放式结局

## 核心设定
- 主角是普通人（上班族/大学生）
- 传说与某个具体地点相关（废弃医院/地铁/公寓）
- 逐渐发现传说与自己有关
- 结局不揭晓最终命运
